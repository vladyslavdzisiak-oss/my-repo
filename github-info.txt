GitHub — це онлайн-сервіс для зберігання Git-репозиторіїв.

Основні можливості GitHub:
- зберігання проєктів у хмарі;
- спільна робота над кодом;
- створення та перевірка Pull Request;
- резервне копіювання репозиторіїв;
- контроль змін та історії проєкту.

GitHub часто використовується для командної розробки програмного забезпечення та зберігання відкритих проєктів.
