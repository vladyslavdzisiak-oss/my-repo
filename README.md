# My Repo

Навчальний проєкт для вивчення Git та GitHub.

## Що буде зроблено

- Робота з локальним Git репозиторієм
- Створення комітів
- Робота з гілками
- Використання .gitignore
- Робота з GitHub
- Використання команд push і pull
- Створення Pull Request
- Клонування репозиторію

## Висновок

У ході виконання роботи було освоєно базові можливості Git та GitHub: створення репозиторіїв, комітів, гілок, злиття змін і роботу з віддаленим репозиторієм.
