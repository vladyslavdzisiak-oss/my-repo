Git — це система контролю версій, яка дозволяє зберігати історію змін у проєкті, створювати коміти та працювати з гілками.

GitHub — це сервіс для зберігання Git-репозиторіїв в інтернеті. Він дозволяє працювати над проєктами спільно, обмінюватися кодом і створювати Pull Request.

Основні переваги Git:
- збереження історії змін;
- можливість повернутися до попередніх версій;
- робота з гілками;
- спільна робота над проєктом.

Основні переваги GitHub:
- резервне зберігання репозиторіїв;
- спільна робота над проєктами;
- обмін кодом;
- керування змінами через Pull Request.
